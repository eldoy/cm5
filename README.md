# cm5
CodeMirror 5 implementation

### Install

```
npm i https://github.com/eldoy/cm5.git
```

### Usage

Require the editor:
```js
var editor = require('editor')
```

Include it on the page and run it:

```html
<textarea id="editor"></textarea>
<script>
  window.editor =${editor}
  editor()
</script>
```
